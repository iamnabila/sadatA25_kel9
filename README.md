Repositori digital digilib.uns.ac.id menyimpan ribuan skripsi mahasiswa Universitas Sebelas Maret, tetapi sistem pencariannya belum mendukung pengurutan hasil berdasarkan relevansi dan tahun terbaru, serta belum menyediakan fitur rekomendasi skripsi serupa. Penelitian ini membangun sistem pencarian skripsi yang menjawab kedua permasalahan tersebut menggunakan struktur data ArrayList, HashMap, dan Graph. Sistem menerapkan Inverted Index untuk pencarian cepat, Merge Sort untuk pengurutan hasil berdasarkan skor relevansi maupun tahun terbit, Jaccard Similarity untuk menghitung kemiripan antara abstrak, dan Breadth-First Search (BFS) untuk menampilkan rekomendasi skripsi serupa. Sistem juga dilengkapi dengan penanganan kata negasi dan perluasan sinonim agar hasil pencarian lebih akurat. Pengujian dilakukan pada 50 data skripsi dalam lingkup Fakultas Teknologi Informasi dan Sains Data. Hasil pengujian menunjukkan sistem mampu mengembalikan  hasil pencarian terurut, relevan, serta memberikan rekomendasi skripsi serupa dengan benar.
